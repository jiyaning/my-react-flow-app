# React Flow Demo

[React Flow 官方文档](https://reactflow.dev/)

## 开发构建

### 快速开始

克隆项目：

```bash
https://github.com/jiyaning/my-react-flow-app.git
```

 进入目录安装依赖：

 ```bash
npm i
 ```

 开发：

```bash
npm run dev
```
